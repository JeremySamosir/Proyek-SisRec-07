# Proyek-SisRec-07

Proyek ini mengevaluasi kemampuan berbagai model Generative AI (LLM) seperti ChatGPT, Gemini, dan Perplexity dalam memahami serta merespons prompt berbahasa Batak. Setiap model dibandingkan berdasarkan keterpahaman, akurasi, kelengkapan, dan relevansi untuk melihat sejauh mana LLM mampu menghasilkan bahasa yang alami dan sesuai konteks budaya Batak, sekaligus mendukung pelestarian bahasa daerah di era digital.

## Tujuan Proyek
1. Mengidentifikasi sejauh mana setiap LLM mampu mengenali struktur kalimat, makna, dan konteks budaya Batak.
2. Mengukur kemampuan model dalam memberikan jawaban yang sesuai dengan konteks sosial, adat, dan nilai-nilai Batak.
3. Memberikan rekomendasi model LLM yang paling efektif untuk dikembangkan lebih lanjut dalam konteks pelestarian atau penerapan bahasa Batak di sistem AI
4. Menentukan model mana yang memberikan respons paling lengkap, alami, relevan, dan menarik dibandingkan model lainnya.

## Model Generative AI yang Digunakan
- ChatGPT 
- Gemini
- Perplexity
